# PaddleOCR Skill：高精度圖片文字識別

## 概述

PaddleOCR 是百度開源的強大光學字符識別 (OCR) 引擎，支持 100+ 語言的文字識別。透過這個技能，Claude 能夠直接從圖片、掃描文件、截圖中提取文字內容，大幅提升圖片處理和數據提取的效率。

**關鍵詞**：OCR、文字識別、圖片轉文本、光學字符識別、多語言、掃描文檔、截圖

## 何時使用此技能

- 📸 從圖片、截圖中提取文字
- 📄 掃描紙質文檔轉換為可編輯的文本
- 🌐 識別多語言混合文本（中英文、日文、韓文等）
- 📊 從表格圖片中提取結構化數據
- 🎫 識別證件、發票、收據上的文字
- 🎨 從設計稿、海報中提取文案
- 📖 掃描書籍或論文進行數字化
- 🔍 分析截圖中的文字內容

## 實施步驟

### 1. 安裝 PaddleOCR
```bash
pip install paddleocr
```

### 2. 基本使用
```python
from paddleocr import PaddleOCR

# 初始化 OCR（自動下載模型）
ocr = PaddleOCR(use_angle_cls=True, lang='ch')

# 識別圖片
result = ocr.ocr('image.jpg', cls=True)

# 提取文字
text = '\n'.join([line[0][1][0] for line in result])
print(text)
```

### 3. 多語言識別
```python
# 識別英文
ocr_en = PaddleOCR(lang='en')

# 識別日文
ocr_ja = PaddleOCR(lang='ja')

# 識別多種語言（自動檢測）
ocr_multi = PaddleOCR(lang='ch_en')
```

### 4. 結果處理
- 提取識別的文字
- 獲取置信度評分
- 提取文字位置坐標
- 過濾低置信度結果

## 精度優化

### ✅ 提高識別準確率的方法

1. **圖片預處理**
   - 提高圖片分辨率（≥ 300 DPI）
   - 增加對比度
   - 去除背景噪音
   - 裁剪到文字區域

2. **參數優化**
   - 根據語言選擇正確的模型
   - 調整 `use_angle_cls`（傾斜文字）
   - 設置合適的縮放倍數

3. **後處理**
   - 使用置信度閾值過濾（例如 > 0.7）
   - 結合 NLP 進行文字修正
   - 手動驗證高風險識別結果

### ⚠️ 常見識別困難

| 場景 | 解決方案 |
|------|---------|
| **手寫文字** | 使用支持手寫的模型，提高圖片質量 |
| **藝術字體** | 增加訓練數據，或使用預訓練模型 |
| **極低對比度** | 圖片增強、調整亮度對比度 |
| **傾斜文字** | 啟用 `use_angle_cls=True` |
| **混合語言** | 使用多語言模型或逐語言處理 |

## 常見問題

### Q: PaddleOCR 支持離線使用嗎？
**A**: 是的。首次使用時會下載模型（約 100-200MB），之後無需網路連線即可使用。

### Q: 如何識別手寫文字？
**A**: 基礎版本主要針對印刷文字。對於手寫文字，建議：
- 使用高分辨率圖片
- 考慮使用專門的手寫識別模型
- 使用 `use_angle_cls=True` 參數

### Q: 識別速度如何？
**A**: 在 CPU 上，識別一張 1080p 圖片通常需要 1-3 秒。支持批量處理和並行推理。

### Q: 如何提高識別準確率？
**A**: 
1. 確保圖片清晰（≥ 300 DPI）
2. 選擇正確的語言模型
3. 使用置信度過濾
4. 對識別結果進行手動審核

### Q: 是否支持掃描的身份證或證件？
**A**: 是的，但建議：
- 使用高質量掃描（不是手機拍照）
- 確保文件完整和清晰
- 對敏感信息的識別結果進行驗證
