---
description: 收官流程：盤點任務、驗證、記錄決策、輸出交付摘要
---

對本次 session 的工作跑收官三步（移植自 Vault for Founders 的 after-action SOP）：

1. **盤點**：表格列出本次所有任務與狀態（✅ 完成 / ⏳ 待使用者 / ❌ 未完成），未完成的說明卡點
2. **驗證**：
   - 跑 `python3 scripts/check-links.py`
   - 檢查中英 README 是否同步
   - 檢查有無未提交的變更（`git status`），有就確認是否該提交
3. **記錄與交付**：
   - 關鍵決策與教訓用表格總結（決策 / 理由 / 影響）
   - 輸出交接清單：下次 session（或使用者本人）要接手的事，按優先級排序

原則：做了不記 = 沒做。全程繁體中文、結論先行。
