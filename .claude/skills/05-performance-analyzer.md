# Skill 5: Performance Analyzer

## 說明
性能分析器，能秒出慢代碼。

## 功能
- 分析代碼中的性能瓶頸
- 如循環嵌套、重複數據庫查詢、大對象創建等
- 給出具體優化建議

## 使用方式
```bash
claude "分析OrderService.java，找出可能的性能問題，並給出優化建議"
```

## 優點
- 快速定位常見性能隱患
- 給出具體優化方案
- 降低性能排查門檻

## 缺點
- 無法檢測運行時性能（需要配合profiler工具）
- 對分布式系統性能問題分析有限

## 適用場景
- 代碼性能優化
- 線上故障排查
- 代碼審查
