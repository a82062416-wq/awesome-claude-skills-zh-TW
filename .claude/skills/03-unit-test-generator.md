# Skill 3: Unit Test Generator

## 說明
單元測試生成器，讓你告別手寫測試。

## 功能
- 基於業務代碼自動生成完整的單元測試用例
- 覆蓋邊界條件和異常場景
- 一鍵快速生成

## 使用方式
```bash
claude "為UserService.java中的createUser方法生成完整的JUnit 5單元測試，包括正常場景、異常場景和邊界條件"
```

## 優點
- 節省大量寫測試的時間
- 提高測試覆蓋率
- 自動生成邊界測試

## 缺點
- 生成的測試可能過於簡單
- 需要人工補充複雜業務場景的測試
- 對複業務邏輯的理解有限

## 適用場景
- 新功能測試、回歸測試用例編寫
- 提高代碼覆蓋率
- 重構前的質量保證
