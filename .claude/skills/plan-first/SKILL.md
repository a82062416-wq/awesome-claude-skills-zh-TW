# 計畫先行（Plan First）

## 啟動條件

- 任務需要動 3 個以上檔案，或步驟超過 3 步
- 任務可能跨多輪對話（需要斷點續作）
- 使用者說「幫我規劃」「這是個大工程」

## 協議

1. **先盤點**：列出會動到的檔案與影響範圍，不確定的先讀再說
2. **建任務清單**：用 TaskCreate 建立步驟（或在對話中列表），每步一個可驗證的完成條件
3. **分階段執行**：一次做一個階段，完成即更新狀態；發現計畫錯誤先改計畫再繼續
4. **每階段驗證**：改動後跑對應檢查（連結檢查、中英同步、格式），不累積到最後
5. **收尾**：對照計畫逐項確認、跑 `python3 scripts/check-links.py`、輸出交付摘要（做了什麼/怎麼驗證/剩什麼）

## 斷點續作

若任務會跨 session：把計畫與進度寫進檔案（如 `tasks/PLAN-主題.md`），
下次 session 開頭先讀該檔案再繼續，避免重新推導。
