# Skill 11: Code Smell Detector

## 說明
代碼異味檢測器，識別低質量代碼模式。

## 功能
- 檢測Long Method、Duplicate Code等代碼異味
- 識別過度耦合和低內聚
- 建議代碼改進

## 使用方式
```bash
claude "掃描src目錄，檢測代碼中的Long Method、Duplicate Code、God Class等代碼異味"
```

## 優點
- 及早發現代碼質量問題
- 指導代碼改進方向
- 提高團隊代碼意識

## 缺點
- 對業務邏輯的理解有限
- 可能存在誤報

## 適用場景
- 代碼質量評估
- 技術債識別
- 代碼改進優先級排定
