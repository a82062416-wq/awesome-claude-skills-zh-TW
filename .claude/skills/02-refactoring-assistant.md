# Skill 2: Refactoring Assistant

## 說明
重構助手，能一鍵優化冗長代碼。

## 功能
- 自動識別代碼環味道
- 提供重構建議
- 甚至可以幫助直接執行重構

## 使用方式
```bash
claude "分析UserService.java，找出重複代碼塊，建議提取公共方法，並給出重構後的代碼"
```

## 適用場景
- 代碼優化和技術債清理
- 代碼規範統一
- 代碼規範統一

## 優點
- 快速定位代碼環味道
- 提供專業重構建議
- 大幅降低低重構門檻

## 缺點
- 自動執行重構有風險
- 需要先分支上測試
- 對框架特定模式的重構可能不準確
