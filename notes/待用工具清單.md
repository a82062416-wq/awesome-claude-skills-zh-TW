## 🎬 影片製作

### video-autopilot-kit（[Hao0321](https://github.com/Hao0321/video-autopilot-kit)）
- **是什麼**：Python 影片自動化框架（ffmpeg pipeline + CapCut 自動化 + 自動字幕 + 交付前 QA）
- **安全**：✅ 765⭐、MIT、v0.8.0（2026-07 更新）、無憑證無 hook
- **何時用**：要**經營頻道／大量產片**時很讚；偶爾剪示範影片則用倉庫的 ffmpeg-video-skill 就夠
- **門檻**：Path 1 純 ffmpeg（跨平台）；Path 2 需 CapCut + Claude Desktop Computer Use
- **和既有的關係**：ffmpeg-video-skill / remotion-video-skill / video-downloader 的批次自動化升級版

## 🖥️ Claude Code 工具

### Claude HUD（[jarrodwatts/claude-hud](https://github.com/jarrodwatts/claude-hud)）
- **是什麼**：Claude Code 狀態列插件，context 余量做成輸入框下進度條
- **安全**：✅ 26.2k⭐、純本機無網路
- **何時用**：想即時看 context/token 用量時，在**自己電腦的 Claude Code** 裝
- **安裝小抄**：見 `notes/claude-hud-安裝小抄.md`（三步 + Linux TMPDIR 注意）
