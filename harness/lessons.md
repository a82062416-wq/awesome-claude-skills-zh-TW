## 教訓

- 2026-07-07｜技能載入｜用 SKILL.zh-TW.md 命名 → 必須同時有 SKILL.md｜載入器只認標準檔名，34 個技能因此死了半年沒人發現
- 2026-07-07｜hooks 設定｜自創 "if" 欄位 → 條件寫進 command 內部判斷｜hooks schema 沒有的欄位會被靜默忽略，不會報錯
- 2026-07-07｜MCP 排程｜重試 create_trigger/send_later 5 次 → 第 2 次失敗就停｜「requires approval」是平台層阻斷，重試不會變
- 2026-07-07｜外部倉庫｜想直接 clone 範圍外 repo → WebFetch 審查後原生實作｜供應鏈信任問題可以用「讀規格重寫」歸零
- 2026-07-07｜WebFetch｜要求長檔逐字輸出 → 只能拿到摘要｜WebFetch 是小模型轉述，短檔才可靠
- 2026-07-07｜驗證閉環｜以為 marketplace 條目都有效 → validate 抓出幽靈條目｜註冊表和實體檔案會漂移，只有機器檢查抓得住
- 2026-07-07｜shell 輸出｜ls 多目錄 + head 截斷 → 誤判 xlsx 是空殼｜管線截斷輸出時先確認完整性再下結論
