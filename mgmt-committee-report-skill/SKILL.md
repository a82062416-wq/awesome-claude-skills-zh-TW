# management-committee-report

## 報告結構
1. 財務摘要（本月收支、管理費收繳率、基金餘額）
2. 工程與維護（進行中/已完成/待處理）
3. 公共事務與住戶反映
4. 決議事項與待追蹤
## 資料來源
接 community-fee-reconciliation 的對帳結果與 changhe-audit 的財報，避免重複輸入。
